## Development

###  Commit message convention


|       Type         |description                          |   
|----------------|-------------------------------|
|feat|새로운 기능에 대한 커밋|
|bugfix|버그 수정에 대한 커밋         |
|build|빌드 관련 파일 수정에 대한 커밋|
|docs|문서 수정에 대한 커밋|
|refactor|코드 리팩토링에 대한 커밋|
|test|테스트 코드 수정에 대한 커밋|
